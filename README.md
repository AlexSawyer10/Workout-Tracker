Workout Routine Tracker
Designed by: Alex Sawyer
Overview
This application is a Workout Routine Tracker that allows you to dynamically add, delete, and edit exisiting workouts. You can create workouts with it being timer based, or rep based. Upon submission of a valid workout, you can view and run your workouts! You will see your workouts are completed when a green bootstrap class goes around your workout.

Functionality Implemented:
✅ Visual Input validation
✅ SetInterval and SetTimeout Functionality
✅ Local Storage
✅ Ecma Script Moduels
✅ Multi-page functionality
